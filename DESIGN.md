# 卒業チキンレース ～目指せ！省エネ124単位～

## コンセプト
大学4年生が卒業必要単位(124)ギリギリで省エネ卒業を目指す対戦カードゲーム。
相手の足を引っ張り合いながら、ピッタリ124単位を目指すチキンレース。

## ルール
- **開始**: 94単位スタート（残り30単位で卒業）
- **ターン**: 全8ターン（交互、各プレイヤー4回ずつ）
- **毎ターン**: 1枚ドロー → 手札から1枚「使う」or「パス」
- **カードの二面性**: キープ=単位として加算 / 使用=イベント効果発動
- **最終計算**: 94 + 手札キープ値合計 + イベント増減

## 勝敗判定
1. 124単位以上で卒業。卒業者が勝ち
2. 両者卒業 → 124に近い方が勝ち（ギリギリ王）
3. 両者不卒業 → 124に近い方が勝ち

## エンディング
| 条件 | エンディング |
|------|-------------|
| ピッタリ124 | 伝説の省エネ卒業（「天才か」） |
| 124+ & ボランティア使用 | 超優良企業へ就職 |
| 131以上 | ガリ勉・友達ゼロ |
| 124-130 | 普通の卒業 |
| 94-123 | 留年（「もう一年遊べるドン！」） |
| 84以下 | 中退（「YouTuberになります」→既読スルー） |
